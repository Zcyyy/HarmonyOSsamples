# ListSample

### 简介
本示例实现了列表内容展示。

### 相关概念
列表内容展示页面主要由四部分组成，从上至下依次是标题、搜索框、列表内容、选项卡。<br>
实现方式：
- 标题：使用Navigation组件实现。
- 搜索框：使用Search组件实现。
- 列表内容：使用List组件实现。 
- 选项卡：使用Tabs组件实现。

### 使用说明
1. 点击底部五个选项卡可以切换到不同的页面。
2. 点击列表内容可以跳转到详细内容页面。
3. 点击搜索框可以输入需要搜索的信息。
